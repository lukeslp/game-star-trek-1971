# Credits and Attribution

## Original Game

**Star Trek (1971)**
- Created by: Mike Mayfield
- Platform: SDS Sigma 7 mainframe
- Language: BASIC
- Date: October 20, 1972 (HP 2000C version)

Mike Mayfield created the original Star Trek game while in high school, inspired by the Star Trek television series and the game Spacewar!. The game was designed to run on a teletype terminal without graphics, using only text-based displays.

## Super Star Trek (1974)

**Enhanced Version**
- Created by: Bob Leedom
- Enhancements: Added damage system, improved combat, library computer functions
- Published in: "The Best of Creative Computing" and "101 BASIC Computer Games"

## Historical Contributors

**David H. Ahl**
- Ported the game to DEC BASIC-PLUS (1973)
- Published in Digital Equipment Corporation's Edu newsletter
- Later published in Creative Computing magazine
- Instrumental in popularizing the game

**Other Contributors**
- Mary Cole (DEC port collaboration)
- Countless programmers who ported and enhanced the game across different platforms throughout the 1970s-1990s

## This Remake (2025)

**Modern Web Implementation**
- Faithful recreation of classic gameplay mechanics
- Enhanced with modern GUI while preserving retro terminal aesthetic
- Built with: React 19, TypeScript, Tailwind CSS 4
- Maintains authentic command-line interface experience

## Design Philosophy

This remake aims to:
- Honor the original game's design and mechanics
- Preserve the authentic text-based command interface
- Add visual enhancements that complement rather than replace the classic experience
- Make the game accessible to modern players while respecting its historical significance
