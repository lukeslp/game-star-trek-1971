# Contributing to Star Trek 1971

Thank you for your interest in contributing to this recreation of the classic 1971 Star Trek game!

## How to Contribute

I welcome contributions that enhance the game while maintaining its classic feel. Here are some ways you can help:

### Bug Reports

If you find a bug, please open an issue with:
- A clear description of the problem
- Steps to reproduce the issue
- Expected vs. actual behavior
- Browser and version information

### Feature Suggestions

I'm open to suggestions that:
- Enhance gameplay while respecting the original 1971 design
- Improve accessibility
- Add quality-of-life features
- Optimize performance

### Code Contributions

1. **Fork the repository**
2. **Create a feature branch**: `git checkout -b feature/your-feature-name`
3. **Make your changes**
4. **Test thoroughly** in multiple browsers
5. **Commit with clear messages**: `git commit -m "Add feature: description"`
6. **Push to your fork**: `git push origin feature/your-feature-name`
7. **Open a Pull Request**

### Code Style

- Use consistent indentation (2 spaces for JavaScript/React)
- Write clear, descriptive variable names
- Add comments for complex logic
- Follow existing code patterns

### Testing

Before submitting a PR:
- Test all game mechanics (movement, combat, docking, etc.)
- Verify the game works in Chrome, Firefox, and Safari
- Ensure the UI is responsive
- Check that all commands function correctly

### Areas for Contribution

**High Priority**:
- Bug fixes
- Accessibility improvements
- Performance optimizations
- Mobile responsiveness

**Medium Priority**:
- Sound effects (optional, toggleable)
- Additional visual polish
- Save/load game state
- Statistics tracking

**Low Priority**:
- Easter eggs
- Achievement system
- Difficulty modes

### What I'm Not Looking For

To preserve the classic experience:
- Major gameplay changes that deviate from the 1971 version
- Multiplayer features
- 3D graphics
- Microtransactions or ads

## Development Setup

```bash
# Clone the repository
git clone https://github.com/lukeslp/game-star-trek-1971.git
cd game-star-trek-1971

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

Feel free to open an issue for any questions about contributing.
